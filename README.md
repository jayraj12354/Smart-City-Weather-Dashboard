# Smart Weather Dashboard

A modern and responsive weather application that provides real-time weather updates with a clean user interface, smart suggestions, and dark mode support.

---

## Features

* Search weather by city name
* Real-time temperature and condition display
* Smart suggestions (clothing and umbrella advice)
* Dark and Light mode toggle
* Fast and responsive interface
* Local storage support for user preferences

---

## Tech Stack

* HTML5 – Structure
* CSS3 – Styling (Glassmorphism and Gradients)
* JavaScript (Vanilla JS) – Logic and API handling
* OpenWeather API – Weather data
