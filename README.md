# Smart City Weather Dashboard

Smart City Weather Dashboard is a web-based application that provides real-time weather information and forecasts for different cities. The project combines multiple weather features into a single, user-friendly interface, simulating a smart city environment.

---

## Overview

This application allows users to search for any city and view current weather conditions along with forecast data. It is designed with a clean UI and focuses on delivering fast and accurate weather updates using external APIs.

Weather dashboards typically allow users to view current conditions and multi-day forecasts for any city through API integration. 

---

## Features

* Search weather by city name
* Display real-time weather data
* Show temperature, humidity, wind speed, and pressure
* Forecast data for upcoming days
* Responsive design for different screen sizes
* Simple and clean user interface

---

## Tech Stack

* HTML
* CSS
* JavaScript
* Weather API (such as OpenWeatherMap)
